Cloud Foundry setup in AWS
###########

###########
Cloud Foundry setup in Windows
###########
1. Install cygwin
2. Install Ruby
3. Go to Ruby folder/bin then execute "gem.bat install bosh_cli --no-ri --no-rdoc" from command prompt
